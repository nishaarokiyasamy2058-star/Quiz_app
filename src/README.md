# AI Quiz Application

A React-based quiz application with:

- Countdown Timer
- Question Palette
- Multiple Choice Questions
- Hint System
- Score Calculation
- Detailed Analysis
- Backend Data Export

## Technologies Used

- ReactJS
- JavaScript
- CSS
- Fetch API

## Features

- 60-second countdown timer
- Previous / Next navigation
- Submit quiz
- Detailed result analysis
- Export quiz results to backend

## Installation

```bash
npm install
npm start
```
